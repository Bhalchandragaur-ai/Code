#include<stdio.h> int main() { char ch; printf("Enter a Character:"); scanf("%c", &ch); if ((ch>='a' && ch<='z') || (ch>='A' && ch<='Z')) printf("It is an Alphabet"); else if (ch>='0' && ch<='9') printf("It is a digit"); else printf("It is a Special Character"); return 0; }

üìö Program Explanation

Ye program user se ek character input leta hai aur check karta hai ki wo:

1. Alphabet hai: Agar character 'a' se 'z' ya 'A' se 'Z' ke beech me hai
2. Digit hai: Agar character '0' se '9' ke beech me hai
3. Special Character hai: Agar digits or alphabets me se kuchh nahi hai
